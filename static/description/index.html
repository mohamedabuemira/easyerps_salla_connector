Important Notes

  • If you have more than one database you should add db_name="your database name" in odoo.conf.

Configuration

1

Odoo Settings

after install the module go to settings >> Users & Companies >> Companies >> and select the company that you want to integrate with Salla

  • Active is Salla shop
  • Go to Salla connection Tap
  • Click Generate API Key

2

Salla App installation and Settings

Go to your store on Salla and install EasyERPS Salla Connector and give the app authorisation

  • Active the app and set the settings parameters
  • Odoo company ID
  • Odoo URL
  • Port
  • API Key that you generated on Odoo
